Reuso

É importante trabalhar com os alunos a importância das funções com uma ferramenta para re-aproveitamento do código.

Prenda-se a Funções são uma espécie de “mini-programa” dentro do código que permite executá-lo facilmente. Além de utilizar demonstre que podemos construir as nossas próprias funções, a linguagem Python possui uma palavra reservada para tal: def.
