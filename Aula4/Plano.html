Aula 4: Texto & Funções¶

Objetivos¶

  • Aprender a trabalhar com textos no Python.
  • Aprender a utilizar algumas funções embutidas da linguagem.
  • Conhecer os caracteres especiais (Unicode)
  • Aprender o que são funções.

Preparação¶

Materiais

  • IDLE - Veja na seção de ajuda Como instalar o Python
  • Editar e executar códigos no IDLE.

Atividades¶

Comece a aula fazendo uma revisão da aula anterior, resgatando o comando print. Veja se eles lembram como o print funciona, senão faça uma revisão. Utilize o exemplo aula4-exemplo1.py como idéia para retomar o conteúdo.

Lembretes

As vezes faz-se necessário explicar com mais detalhes trechos de código, por exemplo, qual motivo de estarmos declarando uma variável chamada ano? É uma boa pratica de programação definir nomes de variáveis capazes de contextualizar seu conteúdo, contudo nem sempre isso é possível e definir nomes de variáveis exageradamente grandes pode criar problemas de legibilidade do código. Para tal a linguagem Python permite escrever comentários, ou seja, blocos de códigos que serão ignorados pelo computador na hora de transformas o código em linguagem de máquina, existem dois tipo:

  • Para comentários de uma única linha, utilizamos o caractere #
# define a idade do jogador
idade = 10
  • Para comentários de múltiplas linhas, utilizamos """ ou '''
"""
Este é um exemplo de comentário
em várias linhas, e o que esta dentro deste bloco
não será executado, a variável abaixo não será definida.
idade = 10
"""

Bônus

Como declarar diversas variáveis com uma única linha de código?
Apenas separe cada uma com , e depois do = atribua o conteúdo respectivo.
